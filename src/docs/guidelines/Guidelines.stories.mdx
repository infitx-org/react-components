# Welcome to @modusbox/react-components

---


#### This library is a collection of reusable React components for building interactive UIs.

### Installation
```shell
yarn add @modusbox/react-components
```

### Usage
Import the components
```tsx
import { Button } from '@modusbox/react-components';

function App () {
  return <Button label="Click Me!"/>
}
```

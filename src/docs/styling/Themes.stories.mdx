Here you can find the color themes used in the components.
Adopting the react-components themes is easy!

- [Kinds](#kinds)

## Kinds

Kinds are used to display the variants.

Available kinds are `primary` `secondary` `tertiary` `success` `danger` `warning` `dark` `light`.

## Colors

Sets of 5 colors for each of the main rainbow colors.
 
Available colors are `blue` `dark-blue` `green` `red` `orange` `gray` `dark-gray` `alpha`.
